# 📱 QR-страница для отзывов

## Описание

Создана скрытая страница `/qr` для размещения QR-кода в салоне.

## Доступ

**URL:** `https://ваш-домен.ru/qr`

Страница не отображается в меню сайта (ни в Header, ни в Footer) и доступна только по прямой ссылке или через QR-код.

## Содержимое

Три красивые кнопки для оставления отзывов:

1. **На нашем сайте** - ведет на `/reviews` (страница отзывов на сайте)
2. **В группе ВКонтакте** - ведет на `https://vk.com/club235192928`
3. **На Яндекс Картах** - ведет на страницу организации в Яндекс Картах

## Настройка

### Обновите ссылку на Яндекс Карты

В файле `app/qr/page.tsx` найдите строку:

```tsx
href="https://yandex.ru/maps/org/cabinet_310/1234567890"
```

Замените на реальную ссылку вашей организации в Яндекс Картах.

### Как найти ссылку на Яндекс Карты:

1. Откройте [Яндекс Карты](https://yandex.ru/maps/)
2. Найдите вашу организацию
3. Скопируйте ссылку из адресной строки
4. Вставьте в код

## Создание QR-кода

1. Перейдите на любой генератор QR-кодов (например, [qr-code-generator.com](https://www.qr-code-generator.com/))
2. Введите URL: `https://ваш-домен.ru/qr`
3. Скачайте QR-код
4. Распечатайте и разместите в салоне

## Рекомендации по размещению

- Разместите QR-код на видном месте (стойка администратора, зеркало)
- Добавьте призыв к действию: "Оставьте отзыв - отсканируйте QR-код"
- Можно предложить небольшую скидку за отзыв

## Дизайн

Страница выполнена в едином стиле с остальным сайтом:
- Градиентный фон
- Декоративные звездочки
- Стеклянные карточки (glass effect)
- Адаптивный дизайн для мобильных устройств
- Анимации при наведении
