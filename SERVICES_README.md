# Страницы услуг — SEO-оптимизация

## Созданные страницы

### 1. `/services` — Главная страница услуг
- **URL**: `/services`
- **Назначение**: Обзор всех услуг студии красоты
- **SEO-оптимизация**:
  - Мета-теги с ключевыми словами
  - Структурированная навигация
  - Хлебные крошки
  - Внутренние ссылки на детальные страницы

### 2. `/services/permanent-brows` — Перманентный макияж бровей
- **URL**: `/services/permanent-brows`
- **Назначение**: Детальная страница услуги перманентного макияжа бровей
- **SEO-особенности**:
  - Расширенные мета-теги (title, description, keywords)
  - Open Graph и Twitter Card
  - Структурированные данные JSON-LD
  - Каноническая ссылка
  - Хлебные крошки для навигации

## SEO-оптимизация

### Ключевые слова
- **Основные**: "перманентный макияж бровей сыктывкар", "пудровые брови сыктывкар"
- **Дополнительные**: "татуаж бровей", "волосковая техника", "кабинет 310 брови"
- **Локальные**: "сыктывкар", "республика коми"

### Структурированные данные
Добавлены JSON-LD разметки:
- Service schema для услуги
- LocalBusiness для организации
- OfferCatalog для прайс-листа
- AggregateRating для рейтингов

### Технические улучшения
1. **Sitemap**: Обновлен с новыми страницами
2. **Navigation**: Добавлены ссылки в Header
3. **Breadcrumbs**: Компонент для навигации
4. **Internal linking**: Связи между страницами

## Структура файлов

```
app/

│       └── structured-data.json   # Структурированные данные

```
