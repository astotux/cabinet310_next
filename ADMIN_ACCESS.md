# Доступ к админ-панели

## Секретный URL для входа

Страница входа для администраторов доступна по адресу:

```
https://cabinet310.ru/admin/secure-entry
```

⚠️ **Важно:** Не делитесь этим URL публично и не добавляйте ссылки на него на сайте.

## Как это работает

1. Обычные пользователи не знают о существовании этой страницы
2. URL не индексируется поисковыми системами (настроено в robots.txt)
3. URL не указан в sitemap.xml
4. После входа работает обычная защита через JWT токены и middleware

## Дополнительная безопасность

Если нужна еще большая защита, можно:
- Добавить IP whitelist в middleware
- Использовать 2FA (двухфакторную аутентификацию)
- Добавить rate limiting для защиты от брутфорса
- Использовать VPN для доступа к админке

## Текущие учетные данные

Проверьте файл `.env` для актуальных данных входа.
