# Настройка уведомлений VK

Система автоматически отправляет уведомления в VK при:
- Новой записи (бронировании)
- Новом отзыве на модерации

## Шаги настройки

### 1. Создайте сообщество VK (если еще нет)
- Перейдите на vk.com/groups
- Создайте новое сообщество или используйте существующее

### 2. Получите токен доступа

1. Перейдите в настройки сообщества → "Работа с API"
2. Создайте ключ доступа с правами:
   - `messages` (отправка сообщений)
3. Скопируйте полученный токен

### 3. Включите сообщения сообщества

1. В настройках сообщества → "Сообщения"
2. Включите "Сообщения сообщества"
3. Разрешите отправку сообщений от имени сообщества

### 4. Получите ID администраторов

1. Откройте профиль администратора в VK
2. ID находится в URL: `vk.com/id123456789` (число после `id`)
3. Или используйте сервис: https://regvk.com/id/

### 5. Настройте переменные окружения

Откройте файл `.env` и заполните:

```env
VK_ACCESS_TOKEN="vk1.a.ваш_токен_доступа"
VK_GROUP_ID="123456789"
VK_ADMIN_IDS="admin_id_1,admin_id_2"
```

Где:
- `VK_ACCESS_TOKEN` - токен доступа сообщества
- `VK_GROUP_ID` - ID вашего сообщества
- `VK_ADMIN_IDS` - ID администраторов через запятую (например: "123456,789012")

### 6. Разрешите сообщения от сообщества

Каждый администратор должен:
1. Открыть диалог с сообществом
2. Написать любое сообщение (например, "Привет")
3. Это разрешит сообществу отправлять сообщения

### 7. Перезапустите сервер

```bash
npm run dev
```

## Формат уведомлений

### Новая запись:
```
🎉 Новая запись!

📋 Услуга: Перманент бровей
👤 Мастер: Лиза
📅 Дата: 2024-03-15
⏰ Время: 14:00

👥 Клиент: Иван Иванов
📱 Телефон: +7 (999) 123-45-67
💬 Комментарий: Первый раз
```

### Новый отзыв:
```
📝 Новый отзыв на модерации!

👤 Имя: Мария
⭐⭐⭐⭐⭐ (5/5)
🎨 Услуга: Маникюр

💬 Текст:
Отличная работа! Очень довольна результатом.
```

## Проверка работы

1. Создайте тестовую запись через форму бронирования
2. Оставьте тестовый отзыв
3. Проверьте, что уведомления пришли в VK

## Отключение уведомлений

Чтобы отключить уведомления, удалите или закомментируйте переменные в `.env`:

```env
# VK_ACCESS_TOKEN="..."
# VK_GROUP_ID="..."
# VK_ADMIN_IDS="..."
```

Система будет работать без уведомлений, просто выведет предупреждение в консоль.

## Troubleshooting

### Уведомления не приходят

1. Проверьте, что токен действителен
2. Убедитесь, что администраторы написали сообщение сообществу
3. Проверьте права токена (должны быть `messages`)
4. Посмотрите логи сервера на наличие ошибок

### Ошибка "User authorization failed"

Администратор должен написать сообщение сообществу первым.

### Ошибка "Access denied"

Проверьте права токена доступа - должны быть включены `messages`.
