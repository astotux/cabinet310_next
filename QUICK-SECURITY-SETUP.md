# 🚀 Быстрая настройка безопасности

## Шаг 1: Сгенерируйте хеш пароля

```bash
npm run hash-password "ваш_пароль"
```

Скопируйте полученный хеш (длинная строка, начинающаяся с `$2a$`)

## Шаг 2: Обновите пароль в базе

```bash
npm run db:studio
```

1. Откроется браузер с Prisma Studio
2. Выберите таблицу **Admin**
3. Нажмите на запись вашего администратора
4. Вставьте скопированный хеш в поле **password**
5. Нажмите **Save 1 change**

## Шаг 3: Проверьте вход

1. Откройте `/admin/login`
2. Введите логин и пароль (тот, что вы хешировали)
3. Должен произойти вход в админ-панель

## Готово! ✅

Теперь ваша админ-панель защищена:
- ✅ Пароли хешированы с bcrypt
- ✅ JWT токены для сессий
- ✅ HttpOnly cookies
- ✅ Автоматическая проверка токенов

---

**Для продакшена:** Не забудьте изменить `JWT_SECRET` в `.env`!
